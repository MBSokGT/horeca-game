# 🎮 Улучшенная механика игры - Инструкция

## Что добавлено

### 1. ⌨️ Typewriter эффект для диалогов
**Файл:** `enhanced-mechanics.js`

Создает эффект печатной машинки при отображении текста диалогов.

**Использование:**
```javascript
const typewriter = new TypewriterEffect();
typewriter.type(element, "Текст для печати", 30); // 30 = скорость в мс
```

**Что дает:**
- Более живой диалог
- Игрок читает текст постепенно
- Можно пропустить кликом

---

### 2. 🎭 DISC-типология клиентов
**Файл:** `enhanced-mechanics.js` → `EnhancedClientGenerator`

Каждый клиент теперь имеет DISC профиль:

- **D (Доминантный)** ⚡ - Прямолинейный, ценит скорость
  - Быстро теряет терпение
  - Любит конкретику
  - Высокий бюджет (+20%)

- **I (Влиятельный)** 🎭 - Общительный, эмоциональный
  - Долго терпелив
  - Любит поговорить
  - Средний бюджет (+10%)

- **S (Стабильный)** 🛡️ - Надежный, осторожный
  - Очень терпелив
  - Ценит доверие
  - Консервативный бюджет (-10%)

- **C (Сознательный)** 📊 - Аналитичный, детальный
  - Средне терпелив
  - Хочет характеристики
  - Точный бюджет (100%)

**Визуальные индикаторы:**
- DISC badge в правом верхнем углу
- Цветовое кодирование
- Подсказки по типу клиента

---

### 3. ⏱️ Система терпения клиента
**Визуальный элемент:** Индикатор терпения (верхний правый угол)

**Как работает:**
- Каждый вопрос тратит терпение
- Разные DISC типы теряют терпение по-разному
- При терпении < 20% клиент может уйти

**Восполнение терпения:**
- Показать каталог: +10%
- Хороший вопрос: +5%
- Подобрать релевантный товар: +15%

**Штрафы:**
- Плохой вопрос: -15%
- Нерелевантный товар: -20%
- Слишком много разговоров: -8% за ход (D-тип)

---

### 4. 🤝 Система доверия
**Визуальный элемент:** Шкала доверия (нижний центр)

**Как работает:**
- Стартовое доверие: 50%
- Влияет на вероятность покупки
- Влияет на скидку

**Повышение доверия:**
- Открытые вопросы: +10%
- Активное слушание: +15%
- Релевантные предложения: +20%

**Снижение доверия:**
- Закрытые вопросы: -5%
- Давление: -15%
- Манипуляции: -20%

---

### 5. 💡 Мгновенная обратная связь
**Файл:** `enhanced-mechanics.js` → `InstantFeedbackSystem`

После каждого действия показывает:
- Качество вопроса (отлично/хорошо/плохо)
- Объяснение почему
- Совет как улучшить

**Типы feedback:**
- 🎯 Excellent - Идеальное действие
- ✅ Good - Хорошо
- ➡️ Neutral - Нормально
- ⚠️ Poor - Можно лучше
- ❌ Bad - Плохо

**Адаптация под DISC:**
Один и тот же вопрос получает разную оценку в зависимости от типа клиента!

---

### 6. 🎯 Умный подбор оборудования
**Функция:** `calculateEquipmentMatch()`

Товары в каталоге теперь сортируются по релевантности:

**Факторы матчинга:**
- Соответствие бюджету (40%)
- Соответствие типу бизнеса (30%)
- Приоритеты клиента (20%)
- DISC предпочтения (10%)

**Визуальные метки:**
- 🟢 High match (>80%) - Зеленая рамка + Badge
- 🟡 Medium match (40-80%) - Желтая рамка + Badge
- ⚪ Low match (<40%) - Полупрозрачность

## 🚀 Как использовать

### Базовый флоу улучшенной игры:

1. **Клиент появляется**
   - Видим его DISC badge
   - Индикаторы терпения и доверия
   - Приветствие с typewriter эффектом

2. **Задаем вопросы**
   - Выбираем из вариантов
   - Получаем instant feedback
   - Следим за терпением

3. **Открываем каталог**
   - Товары отсортированы по match score
   - Видим релевантность каждого товара
   - Подбираем оптимальное предложение

4. **Делаем предложение**
   - Учитывается trust + patience + DISC
   - Расширенный расчет satisfaction
   - Детальный feedback по результату

## 🎨 Кастомизация

### Изменить скорость печати:
```javascript
// В improved-game-integration.js, строка ~60
this.typewriter.type(dialogText, greeting, 30); // 30 → меньше = быстрее
```

### Настроить терпение клиентов:
```javascript
// В enhanced-mechanics.js, EnhancedClientGenerator
maxQuestions: discType === 'D' ? 3 : discType === 'I' ? 5 : 7
// Измените числа для разных типов
```

### Добавить новые фразы приветствий:
```javascript
// В enhanced-mechanics.js, метод generateGreeting()
D: [
    "Ваша новая фраза для D-типа",
    // ...
]
```
