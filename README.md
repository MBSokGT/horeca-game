# Продаван Года — Симулятор менеджера HoReCa

Визуальная новелла-симулятор продаж профессионального оборудования для сегмента HoReCa (отели, рестораны, кафе, столовые).

🎮 **Играть:** https://mbsokgt.github.io/horeca-game/

---

## О игре

Вы — менеджер по продажам оборудования для HoReCa. За 30 рабочих дней нужно выполнить план продаж, общаясь с клиентами разных психологических типов по системе DISC.

### Геймплей

- **Диалоговая система** — выбирайте тип вопроса под психотип клиента
- **DISC-типология** — 4 типа клиентов (D, I, S, C) с разными реакциями
- **Каталог оборудования** — подбирайте позиции под потребности и бюджет клиента
- **Индикаторы** — следите за терпением клиента и уровнем доверия в реальном времени
- **Случайные события** — акции, конкуренты, задержки поставок
- **Прокачка** — репутация и уровень менеджера растут с каждой сделкой

### Уровни сложности

| Уровень | Цель продаж |
|---------|-------------|
| Лёгкий  | 300 000 ₽   |
| Обычный | 500 000 ₽   |
| Сложный | 800 000 ₽   |

### Типы клиентов (DISC)

- **D — Доминантный**: ценит результат и скорость, не любит долгих вступлений
- **I — Влиятельный**: общительный, покупает эмоционально, нужен энтузиазм
- **S — Стабильный**: консервативен, важны надёжность и гарантии
- **C — Сознательный**: аналитик, требует точных цифр и технических деталей

## Технологии

- Vanilla JavaScript (ES6 классы, без фреймворков)
- CSS3 (анимации, backdrop-filter, CSS-переменные)
- localStorage (сохранения, авторизация)
- GitHub Pages (хостинг)

## Структура проекта

```
├── index.html                  # Главный файл игры
├── modern-game.js              # Базовый класс игры (ModernSalesGame)
├── improved-game-integration.js # Расширенная механика (ImprovedSalesGame)
├── enhanced-mechanics.js       # TypewriterEffect, DISC-генератор, обратная связь
├── modern-style.css            # Основные стили
├── enhanced-styles.css         # Стили DISC-механик и анимаций
├── navigation-styles.css       # Боковое меню и настройки
├── clients.js                  # Типы и генерация клиентов

├── events.js / extended-events.js # Случайные события
├── disc-game.js                # Мини-игра «Определи тип DISC»

├── save-system.js              # Сохранение прогресса
```

## Запуск локально

Просто откройте `index.html` в браузере, или запустите локальный сервер:

```bash
npx serve .
```

Затем откройте http://localhost:3000
